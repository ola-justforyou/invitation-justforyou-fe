## Run Locally

Clone the project

```bash
git clone https://github.com/ola-justforyou/justforyou-fe.git
```

Go to the project directory

```bash
cd justforyou-fe
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run start
```

## Tech Stack

**Client:** React JS, TailwindCSS, Material Icons, Material UI, Axios, React Router, AOS
